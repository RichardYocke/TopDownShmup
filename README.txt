1. To run your project, double-click on the file 
	~/TopDownShmup/TopDownShmup.uproject.
2. When you are prompted to rebuild missing Moduels click Yes. 
3. Right Click the project file after it completes the previous step and click "Generate Visiual Studio Project Flies" or you editor equivalent.
4. To run your project again and fully open it, double-click on the file 
	~/TopDownShmup/TopDownShmup.uproject.
5. Add folder StarterContent from the editor by going to "Content Drawer", clicking on "+ Add", then "Add Feature or Content Pack". Go to the "Content" tab, and select "Starter Content".
6. Before opening the Code project, choose Tools->Refresh Visual Studios Code Project.
